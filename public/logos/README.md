# Logos Folder

Place institution logo files in this folder.

## Naming Convention

Logo files should be named using the institution's key (lowercase, with spaces replaced by hyphens):

- `ucla.png` or `ucla.svg`
- `uc-berkeley.png` or `uc-berkeley.svg`
- `stanford.png` or `stanford.svg`
- `mit.png` or `mit.svg`
- `harvard.png` or `harvard.svg`
- `yale.png` or `yale.svg`
- `princeton.png` or `princeton.svg`
- `airbnb.png` or `airbnb.svg`
- `google.png` or `google.svg`
- `tinder.png` or `tinder.svg`
- `apple.png` or `apple.svg`
- `microsoft.png` or `microsoft.svg`
- `meta.png` or `meta.svg`
- `netflix.png` or `netflix.svg`
- `amazon.png` or `amazon.svg`

## Supported Formats

- PNG (recommended)
- SVG (recommended)
- JPG/JPEG

## Notes

- Only institutions with logos in this folder will be displayed
- The component automatically checks for logo files and only shows available ones

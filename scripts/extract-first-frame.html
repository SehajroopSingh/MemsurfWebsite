Select a video file to extract its first frame as a PNG image.

Instructions:
  1. Select your video file (e.g., veo3.1_with_reference_images.mp4)
  2. Click "Extract First Frame"
  3. Review the preview
  4. Click "Download Image" to save it
  5. Save the image as veo3.1_with_reference_images_placeholder.png in public/videos/

Note: This extracts the raw first frame. You may want to apply chroma key (green screen removal) to match the video appearance.
